This repository includes the code and data for preprint arXiv:2208.11884 by Jungmin Kim, Dayeong Lee, Sunkyu Yu, and Namkyoo Park.

Ubuntu 20.04 LTS

### Prerequisite
Python 3.9.6  
Numpy 1.20.3  
Scipy 1.6.2  
Matplotlib 3.4.2
